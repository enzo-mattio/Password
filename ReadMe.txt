Introduction

Ce code en Python utilise la bibliothèque tkinter pour créer une interface graphique pour un programme de stockage sécurisé de mots de passe. Les mots de passe sont cryptés en utilisant la fonction hashlib.sha256 et stockés dans un fichier json.


Prérequis

Les modules tkinter, customtkinter, random, messagebox, hashlib, et json doivent être installés pour que le code fonctionne correctement.


Fonctionnalités

- Vérification de la sécurité du mot de passe
- Stockage sécurisé des mots de passe cryptés dans un fichier json
- Affichage optionnel de la liste des identifiants de mots de passe


Comment utiliser

- Lancer le code en exécutant python nom_du_fichier.py
- Entrer un identifiant et un mot de passe dans les champs appropriés
-Cliquez sur "Ajouter" pour stocker le couple identifiant/mot de passe dans le fichier json

Optionnel: Cliquez sur "Voir" pour afficher la liste des identifiants de mots de passe stockés.


Conclusion

Ce code est conçu pour offrir une solution simple et sécurisée pour stocker des mots de passe. Il peut être amélioré en ajoutant plus de fonctionnalités de sécurité et de gestion de mots de passe.
